# E2E Tests

- smoke.e2e.js — Flow: app launches and root view is visible (smoke).
- authFlows.e2e.js — Flows: email login to home + search navigation; signup to home; nearby list item opens thrift detail and returns home.
- favoritesFlows.e2e.js — Flow: open Favorites tab and navigate to a favorite store detail.
- profileFlows.e2e.js — Flows: edit profile name and save; logout from edit profile.
- categoriesFlows.e2e.js — Flow: open a category and navigate to category store list.
- contentFlows.e2e.js — Flows: edit existing content and save; delete content.
- searchHistory.e2e.js — Flow: save recent searches, remove one, and clear history.
